## A题感悟
虽然这题没啥难度， 但是我还是花了好长的时间。吸取下教训吧。
下面是总结除出的， 我犯的错误。
1. 接受数据用`while`循环接受数据，比如`while(~scanf("%d %d", &n, &m))`，
因为根本不知道他测试的数据是一个还是一群，之前写题目的时候自动默认为1个了。
2. 在`while()`里面要记得用`memset`将数组里的数据清零， 防止下次循环影响到结果。
很重要。 因为没有清零结果浪费了好长时间。
